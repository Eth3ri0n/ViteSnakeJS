# Snake Project with Vite

A Snake game with Vite

## Tech Stack

**Client:** HTML, CSS/SCSS, JavaScript

**Local Server:** Node

## Run Locally

Clone the project

```bash
  git clone https://github.com/Eth3ri0n/ViteSnakeJS.git
```

Go to the project directory

```bash
  cd ViteSnakeJS
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
